# 🎨 BannerCreator 🖼

# 🤔서비스 기획 의도 ?

블로그 포스팅 때마다 고민되는 썸네일!
취향에 맞게 썸네일을 커스텀하고, 이미지 파일로 다운로드까지!
슬기로운 블로그 포스팅을 위해 꼭 필요한 배너 크리에이터를 제작하였습니다.

[배너크리에이터 바로가기](https://bannercreator.netlify.app/)
